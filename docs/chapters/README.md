## 状态定义

| 状态 | 含义 |
| --- | --- |
| done | 当前阶段目标已经实现并通过基础验证。 |
| partial | 已完成核心能力，但仍有生产欠账。 |
| in_progress | 当前正在实现或收口。 |
| todo | 当前阶段内必须处理，但尚未开始。 |
| deferred | 已识别但推迟到后续阶段处理。 |
| planned | 尚未正式进入。 |
